- **Existing Routes**: 
  - `/` (Home/Home Dashboard)
  - `/canais` (Channel management)
  - `/configuracoes` (Global config)

- **UI Components**: The `src/components/ui` folder contains basic atoms (`button.tsx` and `spotlight-card.tsx`). The ecosystem is lacking standard blocks like Inputs, Modals, Sidebar, and the complex Kanban interfaces described in the PRD.

### 2.2 Aesthetic & Visual State
Currently, the application relies on basic CSS (`globals.css`). The objective is to transition completely to the **Design System Lendária**:
- **Dark Mode only** (`bg-base: #050505`).
- No generic placeholders.
- Heavy use of Glassmorphism (`bg-glass`).
- High-contrast highlights (`primary-base: #7C3AED`) and Mesh Gradients.
